* 0. 이 문서에 대하여 (READ FIRST — for Claude Code)

** 미션
**Radiance** — Bible Worlds 프로그램 내 PoC의 코드네임.
하나의 canonical 3DGS 자산으로 *relight · edit · compose* 를 모두 수행할 수 있음을
실제로 입증하는 PoC. 데모용으로 splat을 두 벌 만드는 것이 아니라, **단일 자산**이
세 능력을 전부 지탱한다는 것을 보인다. 프로젝트의 canonical 원칙
("MediaCenter가 자산을 소유, AI는 author가 아니라 renderer/processor")의 실증이다.

단일 canonical 자산의 구성:
- {3DGS geometry + normal}  ← relight(shading) · compose(occlusion) 공통 기반
- {PBR material: albedo/roughness/metallic + env light}  ← relight, compose 시 공유 조명
- {semantic identity encoding}  ← editing(부위 선택/recolor/제거)
- USD 앵커: 명시적 transform + metric scale  ← compose(좌표/스케일)

핵심 통찰: relight와 compose는 "공유 조명"이라는 한 동전의 양면. 재료가 분해된 splat이라야
씬의 빛을 받아 자연스럽게 합성되고, baked splat은 붙여넣은 티가 난다.

*경로 A (권장): 원형 transforms.json 합성.* 헬퍼는 ~src/poses_turntable.py~ 로 분리(tangle).
#+begin_src jupyter-python :tangle src/poses_turntable.py
import json, math, os
import numpy as np

def look_at_c2w(eye, target=(0,0,0), up=(0,0,1)):
    """카메라→월드(c2w) 4x4. 규약은 로더에 맞춰 검증(NOTE)."""
    eye, target, up = map(lambda v: np.array(v, float), (eye, target, up))
    f = (target - eye); f /= np.linalg.norm(f)
    r = np.cross(f, up); r /= np.linalg.norm(r)
    u = np.cross(r, f)
    c2w = np.eye(4)
    c2w[:3,0], c2w[:3,1], c2w[:3,2], c2w[:3,3] = r, u, -f, eye
    return c2w

def turntable_transforms(image_dir, out_json, radius=3.0, elevation_deg=15.0,
                         fov_x_deg=40.0, img_ext=".png"):
    files = sorted(f for f in os.listdir(image_dir) if f.endswith(img_ext))
    n = len(files); el = math.radians(elevation_deg)
    frames = []
    for i, fn in enumerate(files):
        th = 2*math.pi*i/n
        eye = (radius*math.cos(th)*math.cos(el),
               radius*math.sin(th)*math.cos(el),
               radius*math.sin(el))
        frames.append({"file_path": os.path.join("images", fn),
                       "transform_matrix": look_at_c2w(eye).tolist()})
    with open(out_json, "w") as f:
        json.dump({"camera_angle_x": math.radians(fov_x_deg), "frames": frames}, f, indent=2)
    return n, out_json
#+end_src
